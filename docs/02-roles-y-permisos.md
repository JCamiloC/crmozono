# Roles y Permisos

## Roles del sistema
1. Superadministrador
2. Administrador
3. Agente de ventas

## Superadministrador
- Acceso global
- Configuración de países
- Gestión total de usuarios
- Acceso a todos los leads

## Administrador
- Gestión de leads de su país
- Asignación de agentes
- Configuración de mensajes
- Acceso a reportes del país

## Agente de ventas
- Acceso solo a leads asignados
- Registro de llamadas
- Creación de tareas
- Cambio de estado de leads

## Restricciones
- Un agente no puede ver otros países
- Un admin no puede modificar otros países
