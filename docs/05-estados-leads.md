# Estados de Leads

## Estados sugeridos
- Nuevo
- Contactado
- En seguimiento
- Venta efectiva
- No interesado
- Cerrado

## Regla de tiempo
- Cada estado tiene una fecha_estado
- Tiempo máximo: 5 días
- Al vencer:
  - Se genera alerta
  - Se bloquea cierre automático
