# CRM de Ventas con WhatsApp Business API llamado SuperOzono

## Descripción general
Este proyecto es un CRM de ventas enfocado en la gestión de leads, llamadas y mensajería a través de WhatsApp Business API.

Está diseñado para equipos comerciales multi-país, con roles diferenciados y automatización de procesos de contacto, seguimiento y cierre de ventas.

## Objetivos del sistema
- Centralizar la gestión de leads
- Automatizar mensajes de WhatsApp
- Registrar llamadas y resultados
- Facilitar el seguimiento comercial
- Reducir pérdidas de leads por falta de contacto

## Público objetivo
- Equipos de ventas
- Administradores regionales
- Superadministradores globales

## Tecnologías principales
- Next.js (Frontend)
- Supabase (Auth + Base de datos)
- WhatsApp Business API
- Edge / Serverless Functions
