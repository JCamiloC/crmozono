# Flujo General del CRM

## Flujo principal
1. Lead entra por WhatsApp / Facebook
2. Sistema recibe el lead
3. Se asigna país
4. Se asigna administrador
5. Se asigna agente
6. Se inicia contacto
7. Se registra llamada
8. Se cambia estado
9. Se cierra o se programa seguimiento

## Canales de entrada
- WhatsApp API
- Importación manual
- Mensajería masiva

## Automatizaciones
- Mensaje de bienvenida
- Mensaje post-llamada
- Alertas por tareas vencidas
