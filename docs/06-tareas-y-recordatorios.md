# Tareas y Recordatorios

## Objetivo
Evitar pérdida de leads por falta de seguimiento.

## Funcionalidades
- Crear tarea manual
- Programar recordatorio
- Asociar tarea a lead

## Ejemplos
- Volver a llamar
- Enviar mensaje
- Seguimiento de cotización

## Estados
- Pendiente
- Completada
- Vencida
